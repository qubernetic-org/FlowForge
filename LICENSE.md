## Dependency Licensing Policy

Third-party dependencies used in this project must comply with the following:

### Permitted Licenses
- **MIT** — permissive, no restrictions on commercial/internal use
- **Apache 2.0** — permissive with patent grant
- **BSD** (2-clause and 3-clause) — permissive

### Restricted Licenses (requires review)
- **LGPL** — evaluate on a case-by-case basis (dynamic linking generally acceptable)

### Prohibited Licenses
- **GPL** (v2, v3) — copyleft, incompatible with this project's license
- **AGPL** — network copyleft, incompatible with this project's license
- **SSPL** — incompatible with this project's license

### Before Adding Dependencies
1. Verify the dependency's license is in the **Permitted** list
2. If **Restricted**, review before adding
3. If **Prohibited**, do not use — find an alternative
4. Document all dependencies and their licenses
