# Samples

Example visual programs and use cases.

## Purpose

This directory contains example projects demonstrating:
- Basic PLC patterns (timers, counters, state machines)
- Common industrial automation scenarios
- Best practices for visual PLC programming
- Sample JSON project files

## Planned Examples

- **Simple Timer**: Basic timer implementation
- **Traffic Light**: State machine example
- **Conveyor Control**: Motor control with sensors
- **Tank Fill**: Level control with pumps
- **Pick and Place**: Robotic arm control sequence

*Examples will be added as the visual editor becomes functional*
